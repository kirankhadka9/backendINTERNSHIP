This project is a backend system developed for a backend task.
The system collects sales records of different agents and automatically generates a leaderboard ranking agents based on their total sales amount.
Multiple sales records can exist for the same agent.
The system aggregates all records and ranks agents in descending order of total sales amount.
The leaderboard displays:
Rank (1st, 2nd, 3rd, ...)
Agent Name
Total Sales Amount
Total Number of Sales
The application is designed considering correctness, scalability, and real-world usability.
